# Eu Ajudo (iOS)

Aplicativo desenvolvido inteiramente durante o http://hackinpoa.globo.com/.

## Como compilar o projeto

1. `sudo gem install cocoapods`
2. `pod install`
3. Abrir o Xcode, selecionar um Simulador e mandar rodar.
